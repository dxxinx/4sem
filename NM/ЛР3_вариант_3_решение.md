Дана табличная функция:

| x | 0,75 | 0,80 | 0,85 | 0,90 | 0,95 | 1,00 | 1,05 | 1,10 |
|---|---:|---:|---:|---:|---:|---:|---:|---:|
| y | 0,741 | 0,788 | 0,836 | 0,880 | 0,925 | 0,967 | 1,009 | 1,046 |

Промежуточные аргументы:

```text
x1 = 0,83
x2 = 0,98
```

Шаг таблицы:

```text
h = 0,80 - 0,75 = 0,05
```

## 1. Интерполяционный многочлен 3-й степени

Для построения кубического интерполяционного многочлена выберем четыре узла так, чтобы значение `x = 0,83` было внутренней точкой:

| i | xi | yi |
|---:|---:|---:|
| 0 | 0,75 | 0,741 |
| 1 | 0,80 | 0,788 |
| 2 | 0,85 | 0,836 |
| 3 | 0,90 | 0,880 |

Точка `0,83` находится внутри отрезка `[0,75; 0,90]`, поэтому выбранные узлы подходят.

## 1.1. Многочлен Лагранжа

Интерполяционный многочлен Лагранжа имеет вид:

```text
L3(x) = y0*l0(x) + y1*l1(x) + y2*l2(x) + y3*l3(x),
```

где

```text
li(x) = П (x - xj) / (xi - xj), j != i.
```

Для выбранных узлов:

```text
l0(x) = ((x - 0,80)(x - 0,85)(x - 0,90)) /
        ((0,75 - 0,80)(0,75 - 0,85)(0,75 - 0,90))

l1(x) = ((x - 0,75)(x - 0,85)(x - 0,90)) /
        ((0,80 - 0,75)(0,80 - 0,85)(0,80 - 0,90))

l2(x) = ((x - 0,75)(x - 0,80)(x - 0,90)) /
        ((0,85 - 0,75)(0,85 - 0,80)(0,85 - 0,90))

l3(x) = ((x - 0,75)(x - 0,80)(x - 0,85)) /
        ((0,90 - 0,75)(0,90 - 0,80)(0,90 - 0,85)).
```

После подстановки значений функции:

```text
L3(x) =
0,741 * ((x - 0,80)(x - 0,85)(x - 0,90)) /
        ((0,75 - 0,80)(0,75 - 0,85)(0,75 - 0,90))
+
0,788 * ((x - 0,75)(x - 0,85)(x - 0,90)) /
        ((0,80 - 0,75)(0,80 - 0,85)(0,80 - 0,90))
+
0,836 * ((x - 0,75)(x - 0,80)(x - 0,90)) /
        ((0,85 - 0,75)(0,85 - 0,80)(0,85 - 0,90))
+
0,880 * ((x - 0,75)(x - 0,80)(x - 0,85)) /
        ((0,90 - 0,75)(0,90 - 0,80)(0,90 - 0,85)).
```

В развернутом виде:

```text
L3(x) = -6,6666667*x^3 + 16,2*x^2 - 12,1533333*x + 3,556.
```

Вычислим значение при `x = 0,83`:

```text
L3(0,83) =
-6,6666667*(0,83)^3 + 16,2*(0,83)^2 - 12,1533333*0,83 + 3,556
= 0,817.
```

Ответ:

```text
f(0,83) ≈ 0,817.
```

## 1.2. Многочлен Ньютона с разделенными разностями

Многочлен Ньютона записывается в виде:

```text
N3(x) = y0 + (x - x0)f(x0,x1)
        + (x - x0)(x - x1)f(x0,x1,x2)
        + (x - x0)(x - x1)(x - x2)f(x0,x1,x2,x3).
```

Составим таблицу разделенных разностей.

| Узлы | yi | 1-й порядок | 2-й порядок | 3-й порядок |
|---|---:|---:|---:|---:|
| x0 = 0,75 | 0,741 | 0,94 | 0,20 | -6,6666667 |
| x1 = 0,80 | 0,788 | 0,96 | -0,80 | |
| x2 = 0,85 | 0,836 | 0,88 | | |
| x3 = 0,90 | 0,880 | | | |

Проверим основные разности:

```text
f(x0,x1) = (0,788 - 0,741) / (0,80 - 0,75) = 0,047 / 0,05 = 0,94

f(x1,x2) = (0,836 - 0,788) / (0,85 - 0,80) = 0,048 / 0,05 = 0,96

f(x2,x3) = (0,880 - 0,836) / (0,90 - 0,85) = 0,044 / 0,05 = 0,88

f(x0,x1,x2) = (0,96 - 0,94) / (0,85 - 0,75) = 0,02 / 0,10 = 0,20

f(x1,x2,x3) = (0,88 - 0,96) / (0,90 - 0,80) = -0,08 / 0,10 = -0,80

f(x0,x1,x2,x3) = (-0,80 - 0,20) / (0,90 - 0,75)
= -1,00 / 0,15
= -6,6666667.
```

Тогда:

```text
N3(x) = 0,741
        + 0,94(x - 0,75)
        + 0,20(x - 0,75)(x - 0,80)
        - 6,6666667(x - 0,75)(x - 0,80)(x - 0,85).
```

Вычислим при `x = 0,83`:

```text
N3(0,83) = 0,741
           + 0,94(0,83 - 0,75)
           + 0,20(0,83 - 0,75)(0,83 - 0,80)
           - 6,6666667(0,83 - 0,75)(0,83 - 0,80)(0,83 - 0,85)

= 0,741 + 0,0752 + 0,00048 + 0,00032
= 0,817.
```

Ответ:

```text
f(0,83) ≈ 0,817.
```

Результаты по формулам Лагранжа и Ньютона совпадают, так как это разные формы одного и того же интерполяционного многочлена.

## 1.3. Многочлен 3-й степени в Excel

В Excel нужно построить точечную диаграмму по выбранным четырем узлам:

```text
(0,75; 0,741), (0,80; 0,788), (0,85; 0,836), (0,90; 0,880).
```

Далее:

```text
Вставка -> Диаграмма -> Точечная
ПКМ по точкам -> Добавить линию тренда
Тип: Полиномиальная
Степень: 3
Поставить галочку: Показывать уравнение на диаграмме
```

Должен получиться многочлен:

```text
y = -6,6666667*x^3 + 16,2*x^2 - 12,1533333*x + 3,556.
```

При `x = 0,83`:

```text
y(0,83) ≈ 0,817.
```

Вывод по пунктам 1.1-1.3:

```text
Методы Лагранжа, Ньютона с разделенными разностями и полиномиальная
аппроксимация 3-й степени в Excel дают один и тот же результат, потому что
строится один и тот же единственный интерполяционный многочлен 3-й степени
по четырем выбранным узлам.
```

## 2. Конечные разности

## 2.1. Таблица конечных разностей

Составим таблицу конечных разностей по всем точкам.

| x | y | Δy | Δ²y | Δ³y | Δ⁴y | Δ⁵y | Δ⁶y | Δ⁷y |
|---:|---:|---:|---:|---:|---:|---:|---:|---:|
| 0,75 | 0,741 | 0,047 | 0,001 | -0,005 | 0,010 | -0,019 | 0,035 | -0,066 |
| 0,80 | 0,788 | 0,048 | -0,004 | 0,005 | -0,009 | 0,016 | -0,031 | |
| 0,85 | 0,836 | 0,044 | 0,001 | -0,004 | 0,007 | -0,015 | | |
| 0,90 | 0,880 | 0,045 | -0,003 | 0,003 | -0,008 | | | |
| 0,95 | 0,925 | 0,042 | 0,000 | -0,005 | | | | |
| 1,00 | 0,967 | 0,042 | -0,005 | | | | | |
| 1,05 | 1,009 | 0,037 | | | | | | |
| 1,10 | 1,046 | | | | | | | |

Пример вычисления:

```text
Δy0 = y1 - y0 = 0,788 - 0,741 = 0,047
Δy1 = y2 - y1 = 0,836 - 0,788 = 0,048

Δ²y0 = Δy1 - Δy0 = 0,048 - 0,047 = 0,001
Δ²y1 = Δy2 - Δy1 = 0,044 - 0,048 = -0,004
```

## 2.2. Первая и вторая интерполяционные формулы Ньютона

### Значение при x = 0,83 по первой формуле Ньютона

Первая формула Ньютона применяется для интерполирования вперед. Выберем начальный узел:

```text
x0 = 0,80
```

Тогда:

```text
t = (x - x0) / h = (0,83 - 0,80) / 0,05 = 0,6.
```

Так как `0 < t < 1`, выбор начального узла удобен.

Возьмем конечные разности, начиная с узла `x0 = 0,80`:

```text
y0 = 0,788
Δy0 = 0,048
Δ²y0 = -0,004
Δ³y0 = 0,005
```

Формула:

```text
P3(x) = y0 + tΔy0 + t(t - 1)/2! * Δ²y0
        + t(t - 1)(t - 2)/3! * Δ³y0.
```

Подставим:

```text
P3(0,83) = 0,788
           + 0,6 * 0,048
           + 0,6(0,6 - 1)/2 * (-0,004)
           + 0,6(0,6 - 1)(0,6 - 2)/6 * 0,005

= 0,788 + 0,0288 + 0,00048 + 0,00028
= 0,81756.
```

Ответ:

```text
f(0,83) ≈ 0,81756.
```

Сравнение с пунктом 1.3:

```text
По Excel и кубическому многочлену из п.1.3: f(0,83) ≈ 0,817.
По первой формуле Ньютона:              f(0,83) ≈ 0,81756.
```

Разница:

```text
|0,81756 - 0,817| = 0,00056.
```

Разница мала. Она появилась потому, что в пункте 1 использовались узлы `0,75; 0,80; 0,85; 0,90`, а в первой формуле Ньютона для условия `0 < t < 1` были взяты узлы, начиная с `0,80`.

### Значение при x = 0,98 по второй формуле Ньютона

Вторая формула Ньютона применяется для интерполирования назад. Выберем конечный узел:

```text
xn = 1,00
```

Тогда:

```text
t = (x - xn) / h = (0,98 - 1,00) / 0,05 = -0,4.
```

Значение `t` близко к нулю, поэтому выбранный конечный узел удобен.

Для формулы назад нужны нижние конечные разности:

```text
yn = 0,967
∇yn = Δy при x = 0,95 = 0,042
∇²yn = Δ²y при x = 0,90 = -0,003
∇³yn = Δ³y при x = 0,85 = -0,004
```

Формула:

```text
P3(x) = yn + t∇yn + t(t + 1)/2! * ∇²yn
        + t(t + 1)(t + 2)/3! * ∇³yn.
```

Подставим:

```text
P3(0,98) = 0,967
           + (-0,4) * 0,042
           + (-0,4)(-0,4 + 1)/2 * (-0,003)
           + (-0,4)(-0,4 + 1)(-0,4 + 2)/6 * (-0,004)

= 0,967 - 0,0168 + 0,00036 + 0,000256
= 0,950816.
```

Ответ:

```text
f(0,98) ≈ 0,950816.
```

Для сравнения, если построить естественный кубический сплайн по всем точкам, то:

```text
S(0,98) ≈ 0,950335.
```

Разница:

```text
|0,950816 - 0,950335| = 0,000481.
```

Итоговый вывод:

```text
Интерполяция по многочлену Лагранжа и многочлену Ньютона с разделенными
разностями дает одинаковый результат при использовании одних и тех же узлов.
Интерполяционные формулы Ньютона с конечными разностями также дают близкие
значения. Небольшие расхождения связаны с тем, что для разных формул
выбираются разные наборы ближайших узлов. Чем ближе выбранные узлы к
исследуемой точке, тем надежнее результат интерполяции.
```
